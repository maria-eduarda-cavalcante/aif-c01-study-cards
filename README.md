# AIF-C01 — Study Cards & Mock Exam

Site estático com flashcards em modais e simulado interativo para apoiar os estudos da certificação **AWS Certified AI Practitioner (AIF-C01)**.

## Publicar no GitHub Pages

1. Crie um repositório público (ex.: `aif-c01-study-cards`) e envie os arquivos para a branch `main` na raiz.
2. Adicione os arquivos fornecidos: `index.html`, `.nojekyll`, `404.html`, `README.md` e `LICENSE`.
3. No repositório, vá em **Settings → Pages** e selecione **Deploy from a branch**, escolha **main** e **root**.
4. Aguarde a publicação e acesse `https://<usuario>.github.io/<repo>/` (ex.: `https://seu-usuario.github.io/aif-c01-study-cards/`).

> `.nojekyll` garante que o GitHub Pages trate os arquivos diretamente, e `404.html` redireciona para a raiz para evitar erros ao navegar em subcaminhos.
